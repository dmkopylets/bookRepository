# Book Repository
(Symfony RestAPI) 

**to build the test task program, use commands:**

make dc-build

make dc-up

make bash

cd app

composer update

npm install

php bin/console doctrine:migrations:migrate

exit 

**to start the test task program, use:**

make dc-up


Implemented also: Docker containerization system
